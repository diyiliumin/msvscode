# msvscode

为什么我的电脑里有个叫微软大战代码的游戏啊

## 简介

简单实现微软大战代码

### 安装方法

编译c程序，可以放到userbin下面

### 使用方法

打命令

```bash
vscode          #或者你编译后起的名字
vscode main.c   #可带一个文件名作为参数
```

然后观看微软大战代码即可

有时打架太久，让人失去耐心，可以按q全杀了

之后程序会嗅探你的文本编辑器，如nano，vim之类，优先从你的$EDITOR环境变量中提取，如果没设置，会fallback到你有的编辑器上

然后它会启动这个编辑器，如果你追加参数你就可以编辑这个文件了

所以它甚至有vscode的部分功能对吧（笑
